# Dashboard Data Snapshot

These CSV files are a committed snapshot of the reproducible BI exports in `data/bi_exports/`.

They are included so the Streamlit dashboard can run in a lightweight hosted environment without access to the local DuckDB warehouse or raw source files.

The snapshot includes match predictions, group standings, team profiles, player power profiles, model metrics, data quality checks, historical scoring context, match feature context, player star-power fields, tournament simulation probabilities, and route difficulty.
